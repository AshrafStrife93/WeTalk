Project Title: Chatting Application
By: Saiful Ashraf Bin Sa'adi

Step 1: Install nodejs. Also can get it here: https://nodejs.org/en/
Step 2: Open server folder, double click to run runserver.bat and leave it open.
Step 3: Install WeTalk.apk into your Android mobile phone/Tablet.
Step 4: Run WeTalk from Android mobile phone/Tablet.
Step 5: Enter your nickname and server IP Address together with port 3333. Example: http://192.168.1.5:3333
